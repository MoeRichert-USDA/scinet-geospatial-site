Itinerary

2022 Geospatial Workshop Itinerary

Please review the pre-meeting checklist and background information on the Pre-meeting page to ensure you are prepared for the workshop sessions.


    • Session 0: Pre-meeting technical support for account logins and data uploads

      • Thursday August 25, 10:00am-11:00am MDT
        • Prerequisites: None

    • Session 1: Introduction to SCINet with lightning presentations

      • Monday August 29, 10:00am-11:00am MDT
        • Prerequisites: None

    • Session 2: Fundamentals of geospatial data

      • Monday August 29, 11:00am-12:00pm MDT
        • Prerequisites: None

    • Session 3: Fundamentals of parallel processing

      • Monday August 29, 2:00pm-3:00pm MDT
        • Prerequisites: None

    • Session 4: SCINet interactive environments

      • Tuesday August 30, 10:00am-12:00pm MDT
        • Prerequisites:
          • Have a SCINet account and be able to login
          • Basic understanding of partitions and requesting cores. If you are unfamiliar with these topics, please attend Session 3 on parallel processing fundamentals.

    • Session 5: SCINet Geospatial Common Data Library

      • Tuesday August 30, 12:30pm-1:30pm MDT
        • Prerequisites:
          • Have a SCINet account and be able to login
          • Basic R or other basic programming skill helpful (expertise not required).
          • Interactive environments will be used, so attending Session 4 may be helpful.

    • Session 6: Geospatial analyses and how to parallelize them in R

      • Wednesday August 31, 11:00am-1:30pm MDT
        • Prerequisites:
          • Have a SCINet account and be able to login
          • Basic R or other basic programming skill helpful (expertise not required)
          • RStudio on Ceres via Open OnDemand will be used, so attending Session 4 may be helpful

    • Session 7: Interactive BYOP session + Open Drone Map demo

      • Wednesday August 31, 2:00pm-3:00pm MDT
        • Prerequisites:
          • Have a SCINet account and be able to login

    • Session 8: Geospatial analyses and how to parallelize them in python

      • Thursday September 1, 10:00am-12:00pm MDT
        • Prerequisites:
          • Have a SCINet account and be able to login
          • Basic python or other basic programming skill helpful (expertise not required)
          • JupyterLab on Ceres via Open OnDemand will be used, so attending Session 4 may be helpful

    • Session 9: Interactive BYOP session

      • Thursday September 1, 2:00pm-3:00pm MDT
        • Prerequisites:
          • Have a SCINet account and be able to login

    • Session 10: Interactive BYOP session + TBD special topic based on previous sessions' discussions

      • Friday September 2, 10:00am-12:00pm MDT
        • Prerequisites:
          • Have a SCINet account and be able to login

    • Session 11: Closing discussion and future planning

      • Friday September 2, 12:30pm-1:30pm MDT
        • Prerequisites: None
